About the Directory

  • Search the Companies House Database

    This tool allows you to search for companies registered in the UK using the official Companies House API. Enter your search terms to discover innovative companies in the Cambridge area and beyond.

  • How to Use the API

    You'll need a free API key from Companies House to use this service. Register at the Companies House Developer Portal to obtain your key. Your key is stored securely in your browser session only.

Latest Updates

  • New Features Added

    We've added advanced filtering, sorting options, and multiple view modes to help you find exactly what you're looking for.

  • Data Privacy Information

    Your API key is never stored on our servers. All search data is processed directly with the Companies House API for maximum security.
